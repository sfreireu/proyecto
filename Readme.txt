Primero se instaló el paquete NPM para generar la estructura base del proyecto.

Los formularios están generados con el siguiente orden lógico:

1)login.js el cual debería tener la función de habilitar los formularios según el rol del usuario
2)Formulario.js el cual sería el formulario al que tiene acceso el administrador para poder registrar los empleados
3)FormularioEmpleados.js el cual funcionaría para modificar la información de los empleados
